# 🎨 Design System (fonte única de UI)

> **Regra da IA:** Tokens e padrões de UI deste projeto. Fonte única — nenhum
> valor de cor/espaçamento/tipografia "mágico" espalhado pelo código. Curado pelo
> `designer_ux` + `biblioteca_conhecimento`. Nasce vazio; preencha conforme a UI
> nasce. Não é específico de stack — é o vocabulário visual do produto.

## Tokens
- **Cores:** _paleta + contraste (WCAG AA mínimo)._
- **Tipografia:** _escala, pesos, pareamento._
- **Espaçamento:** _escala base (ex: 4/8px)._
- **Raio / sombra / elevação:** _shape language._
- **Breakpoints:** _mobile-first._

## Componentes-base & estados
_default · hover · active · focus · disabled · error · loading · empty._

## Acessibilidade (mínimos)
_Contraste, foco visível, navegação por teclado, alvos ≥ 44×44px, semântica/ARIA._

## Padrões adotados
_Convenções de navegação, feedback (toasts/alerts), formulários, etc._
