# 🗂️ ADRs — Architecture Decision Records

Decisões arquiteturais relevantes são registradas aqui, uma por arquivo, no
formato `NNNN-<slug>.md` (ex: `0001-escolha-de-banco.md`), a partir de
`templates/adr_template.md`.

Um ADR responde: **qual o contexto, qual a decisão, e quais as consequências.**
A IA escreve um ADR na Fase 5 quando uma decisão tem impacto de longo prazo —
sobretudo ao quebrar/versionar a API pública de um módulo (gate de risco).

Nasce vazio.
